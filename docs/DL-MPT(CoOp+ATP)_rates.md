# DL-MPT 最终实验报告

## 实验设置

- **方法**: DL-MPT (Dual-loop Meta Prompt Tuning) + CoOp_ATP
- **骨干网络**: CLIP ViT-B/16 (frozen)
- **训练**: 25 epochs (Warmup 5ep λ=0 → Joint 15ep λ=0.2 → Refine 5ep λ=0.5)
- **Baseline**: Paper CoOp+ATP (100 epochs, 标准 CE 训练)
- **协议**: Base-to-New (在 Base 类上训练, Novel 类上 zero-shot 评估)
- **种子**: 3 seeds (1, 2, 3)


## 运行命令

```bash
# 训练
bash scripts/dlmpt/train.sh --dataset stanford_cars --seed 1 --lambda 0.2 --n-way 20

# 评估 Novel
python train.py --root DATA --seed 1 --trainer CoOp_ATP \
    --dataset-config-file configs/datasets/stanford_cars.yaml \
    --config-file configs/trainers/CoOp/vit_b16.yaml \
    --model-dir output/dlmpt/stanford_cars/seed1/lambda0.2 --load-epoch 25 --eval-only \
    DATASET.SUBSAMPLE_CLASSES new \
    TRAINER.ATPROMPT.USE_ATPROMPT True TRAINER.COOP.N_CTX 2
```
